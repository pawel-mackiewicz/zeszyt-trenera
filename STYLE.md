# Visual Style Prompt: The Analog Tactician (Utilitarian Ledger)

**Core Aesthetic:** A "no-fluff," rugged, and highly functional mobile tool designed to look like a physical instructor's ledger or technical field manual.

**Background:** Use a subtle, light-gray or off-white graph paper grid (16px or 20px squares) as the primary background texture.

**Borders & Lines:** Utilize sharp, 1px solid black (or very dark zinc) borders for all containers, buttons, and section dividers. No rounded corners (0px radius) unless specifically for a "hand-drawn" accent.

**Typography:** Exclusively use monospaced fonts (e.g., Space Mono, Roboto Mono, or Courier) for data, labels, and secondary text. For main headers, use a bold, high-contrast sans-serif (e.g., Space Grotesk) in all-caps to create a clear visual hierarchy.

**Color Palette:**

- **Primary:** `#000000` (Black) for text, borders, and structure.
- **Background:** `#F9F9F9` (Off-white/Light Gray) with subtle grid lines.
- **Accent:** A vibrant "Ballpoint Pen Red" (`#AE1417`) used sparingly for primary actions, status indicators, and critical data points to draw immediate focus.

**Components & UI Patterns:**

- **Buttons:** Rectangular, flat, with 1px black borders. Active states should feel tactile (e.g., slight 1px-2px offset shift).
- **Navigation:** A fixed, utilitarian bottom navigation bar with clear icon/label pairs, using heavy borders to separate it from the content.
- **Data Presentation:** Present information in a vertical "log" or "ledger" format, using uppercase labels and clear separation lines.

**Mood:** Utilitarian, tactical, analog, precise, and devoid of modern SaaS "softness" (no gradients, no soft shadows, no large border radii).
